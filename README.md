# CMS-K and K-Screens

SE Project

## Instructions
1. Install node (download from website)
2. Install grunt-cli (npm install -g grunt-cli)
3. Type "npm install"

We use MAMP for the local webserver.

## Technology Stack

* [jQuery]
* [Bootstrap]
* [Grunt]
* [PHP]
* [MySQL]
